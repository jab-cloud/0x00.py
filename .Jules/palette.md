## 2025-05-14 - Empty alt text for decorative elements in Jupyter Notebooks
**Learning:** Decorative elements like dividers in Jupyter Notebooks should use empty alt text (`![]()` in Markdown or `alt=""` in HTML) to prevent screen readers from announcing them, which improves the overall accessibility and reduces noise for users. Informative images like logos or schemas should have descriptive alt text.
**Action:** Always audit Jupyter Notebooks for decorative images and ensure they have empty alt text, while informative images have meaningful descriptions.
