## 2024-05-14 - Improve accessibility in Jupyter Notebooks
**Learning:** Decorative elements in Jupyter Notebooks, such as custom dividers, should use empty alt text (e.g., `![]()`) to avoid screen reader noise. Informative images must have descriptive alt text to provide context for visually impaired users.
**Action:** Always audit Jupyter Notebooks for image accessibility using `grep` and ensure proper alt text usage.
