## 2026-05-09 - Accessibility in Jupyter Notebooks
**Learning:** Jupyter Notebooks often contain images (logos, diagrams, decorative dividers) that lack proper accessibility attributes. Informative images like database schemas need descriptive `alt` text, while decorative elements like dividers should have empty `alt` text to avoid screen reader noise.
**Action:** Always check Jupyter Notebook markdown cells for images and ensure they follow accessibility best practices for `alt` text.
